# 🎓 Attendance Analytics SaaS

A production-ready attendance tracking platform with prediction engine and analytics.

## 🚀 Features
- Secure login (Firebase Auth)
- Real-time attendance tracking
- Prediction: attend/miss impact
- Safe bunk calculator
- Subject ranking
- Attendance timeline & heatmap
- Profile photo upload
- Cloud database
- Multi-user SaaS architecture

## 🛠 Tech Stack
React • Firebase • Firestore • Storage
